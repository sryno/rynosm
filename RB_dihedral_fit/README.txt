usage: RBCurveFit.py [-h] INPUT_FILE

Fits a Ryckaert-Bellemans Dihedral to a set of points. Angles should start at
0 or 180.

positional arguments:
  INPUT_FILE  Input text file with ANGLE VALUE on each line. Angle should be
              in degrees.

optional arguments:
  -h, --help  show this help message and exit
